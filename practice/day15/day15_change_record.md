# Day 15 — Prompt 修改记录

## 修改内容

**修改文件：** `backend/agents/nl2sql_agent.py`

**新增规则：**
```
【聚合函数约束】如果用户问的是：
- 平均数 → SQL 必须用 AVG()
- 总数 → SQL 必须用 SUM()
- 计数 → SQL 必须用 COUNT()
```

## 为什么改？

（写下你的理由）

答：LLM 有时会忘记用聚合函数，导致生成的 SQL 没有对数据做汇总。
